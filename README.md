# Decision Ledger System

A platform for organizations to document, track, and analyze decisions with full transparency and accountability. Built with the MERN stack.

## Tech Stack
- **Backend:** Node.js, Express, MongoDB, Mongoose, JWT
- **Frontend:** React, Vite, Tailwind CSS, React Router, React Hook Form

## Features
- **User Roles:** Admin, Decision-Maker, Viewer
- **Decision Tracking:** Complete CRUD for decisions with rich data points (category, outcome, priority)
- **Dashboard:** Filter and search capabilities
- **Responsiveness:** Full mobile-first design
